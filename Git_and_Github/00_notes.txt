Git -> free and open source version control system

Version control system -> tools that helps to track changes in code

Github -> website where we host repositories online

git -version 

git config --global user.name "Giriraj"

git config --global user.email "girirajcodingprofile@gmail.com"

git config --list 

clone -> cloning a repository on our local machine
git clone link

status -> display the state of the code
git status

file status lifecycle -> untracked unmodified modified staged

add -> add new or changed or file in your working directory to the git staging area
git add fileName
for all files -> git add .

commit -> it is the record of change ye commit keval apne local repo me hue hai online github par nahi
git commit -m "some message"


push -> upload local repo content to remote repo
git push origin main


init -> used to create a new git repo
git init
